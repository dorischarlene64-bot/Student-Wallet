# Student Wallet Management System

## Overview

The Student Wallet Management System is a web-based application designed to help university students manage and track their financial transactions efficiently. The system allows users to record expenses, monitor wallet balances, and maintain an organized history of transactions through a user-friendly interface.

## Features

* User Registration and Login
* Secure Authentication System
* Add and Manage Wallet Transactions
* Track Current Balance
* View Transaction History
* Database-Driven Data Storage
* Responsive and User-Friendly Interface

## Technologies Used

### Backend
